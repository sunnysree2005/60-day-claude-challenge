# Day 35: Build Prompt Puzzle 🧩

Welcome to my submission for **Day 35** of the Interactive Prompt Engineering challenge! Today's task focused on building and playing **"Prompt Puzzle"**—an interactive web application designed to master AI prompting techniques through gamified challenges.

---

## 🚀 Project Overview
Prompt Puzzle is a browser-based interactive game where players learn, test, and refine their prompt engineering skills by solving specific AI constraint puzzles across different difficulty levels and domains.

### Key Features:
* **Interactive Challenges:** Play through multiple game levels testing core prompt engineering principles (context setting, constraints, few-shot prompting, etc.).
* **Dynamic Evaluation:** Generates a detailed **Prompt Performance Report** after each round.
* **Replayability:** Ability to switch between different domains and difficulty settings.

---

## 🛠️ Step-by-Step Implementation

1. **Prompt Generation:** Used Claude (with Low effort level) with the official Prompt Puzzle prompt template.
2. **Setup:** Answered the onboarding questions and generated the complete standalone HTML application (`index.html`).
3. **Local Testing:** Opened the HTML file locally in the browser to test all 3 challenges, scoring systems, and feedback reports.
4. **Documentation & Proof:** Captured screenshots of the gameplay and performance report.
