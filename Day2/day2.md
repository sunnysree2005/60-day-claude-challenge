# Day 2 Challenge: Frontend Engineer Application Prompt Template

This file contains the engineered prompt template and the optimized output template used for Day 2 of the Claude AI Mastery Challenge.



## 1. Engineered Prompt (The Input)


Role: Senior Technical Recruiter and Expert Copywriter

Context: I am a recent computer science graduate specializing in frontend development (React, JavaScript, CSS). I want to write a tailored, highly professional application email for a Frontend Engineer position at a target company.

Task: Generate a high-converting job application email template.

Constraints:
1. Clearly highlight foundational skills and passion for building fast, accessible interfaces.
2. Include a structured placeholder to showcase a metric-driven project (e.g., app used by 50+ beta testers).
3. Ensure a professional sign-off with standard contact fields (Phone, LinkedIn, GitHub).

Format: Clean, ready-to-use email template with brackets for customization.
2. Optimized Output Template (The Result)
This is the production-ready template generated by Claude using the engineered prompt above:

Plaintext
Subject: Application for Frontend Engineer Role – [Your Name]

Hi [Hiring Manager's Name],

I'm writing to apply for the Frontend Engineer position at [Company Name]. I'm a recent computer science graduate with hands-on experience in React, JavaScript, and CSS, and a strong passion for building fast, accessible user interfaces.

During my studies, I built [briefly describe a project — e.g., 'a real-time task management app using React and Node.js, used by 50+ beta testers']. I'm excited about [Company Name] because [one specific reason — product, mission, tech stack].

I've attached my resume and would love the opportunity to discuss how I can contribute to your team.

Thank you for your time,
[Your Name]
[Phone Number] | [LinkedIn]
