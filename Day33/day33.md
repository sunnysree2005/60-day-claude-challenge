# 🔍 Media Integrity Analyzer

An interactive, educational single-file simulator designed to teach users how to critically evaluate news, identify cognitive biases, and dissect emotional manipulation in modern media. Developed as part of the **60-Day Coding Challenge (Day 33)**, this application transforms media literacy training into a gamified, high-fidelity diagnostic experience.

---

## 🧭 Application Flow & Core Modules

1. **Theme Initialization:** Custom color engine selector allowing the analyst to configure their preferred visual operational workspace.
2. **Phase 1: Headline Detective:** 
   * Challenges users to evaluate sensationalized, clickbait, or highly biassed headlines.
   * Breaks down the structural anatomy of the text and provides an objective, facts-first rewriting of the news report.
3. **Phase 2: Emotion Detector:**
   * Isolates specific emotional manipulation vectors (e.g., fear-mongering, out-group villainization, loaded language).
   * Teaches the logical fallacies hidden beneath persuasive prose.
4. **Real-Time Telemetry Tracking:** Live-updating structural metrics processing the current analysis pass:
   * **Objectivity Score**
   * **Fact Density**
   * **Bias Mitigation**
   * **Emotional Neutrality**
5. **Media Integrity Dashboard Summary:** An executive analytical overview detailing the user's critical evaluation performance, key takeaways, dynamic risk assessments of randomized media scenarios, and three essential rules for media consumption.

---

## 🎨 System Design & Architecture

* **Modern Dark UI Theme:** Sleek, high-contrast workspace UI built to resemble an analytical command deck, utilizing fluid transitions and micro-interactions.
* **Componentized State Engine:** Handcrafted with React functional hooks (`useState`) to drive multi-tiered analytical gates and step progressions dynamically.
* **Zero Infrastructure Dependency:** Runs completely standalone and works flawlessly offline. Built inside a single HTML file leveraging React and Babel over standard, secure CDN gateways.

---

## 🛠️ Technical Specifications

* **Runtime Support:** Modern Cross-Platform Web Browsers (Chrome, Edge, Firefox, Safari).
* **View Library Layer:** React JS (via CDN)
* **On-the-fly Compiler:** Babel JSX (via CDN)
* **Design Implementation:** Native CSS3 (Custom Properties, Flexbox, CSS Grid, Transitions).
* **Asset Integration:** Completely self-contained asset architecture requiring no external network hits for fonts, scripts, or icons.
