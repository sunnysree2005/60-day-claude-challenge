# Day 36: Cognitive Pattern Explorer 🧠

Welcome to my submission for **Day 36** of the Interactive Prompt Engineering challenge! Today's task focused on building an interactive, psychology-inspired web application called **"Cognitive Pattern Explorer"** using Claude, designed to explore personal thinking tendencies, decision-making patterns, and reactions under stress through gamified scenarios.

---

## 🚀 Project Overview
The Cognitive Pattern Explorer helps users reflect on how they approach decisions and uncertainty through calm, engaging chapters. 

### Key Features:
* **Calm & Stress Modes:** Experience how varying operational contexts shift decision-making styles.
* **Chapter-Based Exploration:** 
  * *Chapter 1:* Interactive everyday scenario responses mapping to distinct thinking styles (Analytical Thinker, Emotional Intuitive, Overthinking Loop Style, Action-First Decision Maker, Balanced Reflective Thinker).
  * *Chapter 2:* Priority sorting utilizing interactive drag-and-drop mechanics.
  * *Chapter 3:* Timeline mapping for cognitive sequences.
* **Personalized Reflection Journal:** Generates a real-time percentage breakdown and insightful, non-clinical feedback summarizing your dominant thinking profile.

---

## 🛠️ Step-by-Step Implementation

1. **Prompt Execution:** Provided the Cognitive Pattern Explorer prompt to Claude to generate a clean, standalone, responsive single-file web app (`index.html`).
2. **Local Testing:** Opened the HTML file locally in the browser to navigate through Calm vs. Stress modes, evaluate drag-and-drop interactions, and verify scoring logic.
3. **Documentation:** Captured screenshots of the Reflection Journal and Thinking Profile breakdown.
