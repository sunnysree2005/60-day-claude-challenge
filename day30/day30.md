# 🌐 Supply Chain Builder Optimizer

An interactive, premium enterprise dashboard simulator designed to teach the fundamentals of supply chain operations. Built as part of the **60-Day Coding Challenge**, this single-file web application allows users to step into the shoes of an Operations Leader to configure, optimize, and evaluate a global supply chain strategy against dynamic constraints.

---

## 🚀 Live Simulation Flow

1. **Onboarding & Introduction:** A welcoming interface introducing core supply chain principles in highly accessible language.
2. **Dynamic Company Generation:** Every playthrough generates a randomized enterprise profile specifying its industry, product portfolio, target regions, and baseline market demand.
3. **Strategic Decision Gates:** A guided, step-by-step wizard walking the operator through critical logistical configurations, complete with transparent trade-offs for each option:
   * **Sourcing Strategy:** Single vs. Multiple Suppliers.
   * **Manufacturing Infrastructure:** Domestic vs. Nearshore vs. Offshore Factory Locations.
   * **Fulfillment Strategy:** Centralized vs. Decentralized Warehouse Networks.
   * **Logistics & Distribution:** Balancing speed and carbon footprint via Road, Rail, Ocean, or Air Freight.
   * **Inventory Management:** Lean (Just-in-Time) vs. Balanced vs. Buffered (Safety Stock) strategies.
4. **Real-Time Key Performance Indicators (KPIs):** Dynamic metric counters monitoring **Cost, Delivery Speed, Risk Profile, Customer Satisfaction,** and **Sustainability** indexes updating instantly with every choice.
5. **Executive Audit Dashboard:** A comprehensive analytical wrap-up yielding an **Overall Supply Chain Score (0–100)**, clear callouts for structural Strengths and Weaknesses, a designated "Top Risk Vulnerability," and three actionable operational recommendations.

---

## 🎨 Design & Architecture

* **Premium Enterprise UI:** High-fidelity, dark-themed responsive layout optimized for analytical visualization.
* **Component-Driven State:** Engineered strictly using reusable React functional components mapping complex state changes across the decision pipeline.
* **Micro-Animations:** Fluid CSS transitions, interactive hover behaviors, and animated performance bars providing modern, instantaneous visual feedback.
* **Zero Overhead Dependencies:** Packaged entirely into a single standalone HTML document using React and Babel via secure CDN pipelines, running reliably out-of-the-box offline.

---

## 🛠️ Tech Stack & Dependencies

* **Runtime Environment:** Standard Web Browser (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari).
* **Core Framework:** React JS (via CDN)
* **Compiler:** Babel JSX (via CDN)
* **Styling Engine:** Raw CSS3 (Custom Variables, Flexbox, Grid, Keyframes).
* **External Assets:** None (Fully self-contained, icon-free typographic interface).
