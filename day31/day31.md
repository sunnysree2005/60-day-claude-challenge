# 📡 AI Supply Chain Control Tower Simulator

An interactive, high-fidelity enterprise control tower dashboard designed to simulate real-world incident response, risk mitigation, and operational crisis management. Built as part of the **60-Day Coding Challenge (Day 31)**, this application casts the user as the **Head of Global Operations**, tasked with monitoring live disruptions and executing corrective actions under pressure to maintain critical KPIs.

---

## 🎮 Simulation Framework & Core Features

1. **Incident Control Center:** A live, dynamic alert ticker generating randomized supply chain disruptions (e.g., geopolitical border closures, severe weather delays, raw material shortages, cyberattacks, or sudden demand spikes).
2. **Triaging & Prioritization:** Forces the operator to evaluate incident urgency, financial impact, and time-critical windows before selecting a mitigation path.
3. **Strategic Corrective Actions:** Multi-option choice architecture for every event—allowing tactical pivots such as expedited air freight, switching to secondary suppliers, drawing from safety stock, or re-routing maritime cargo.
4. **Real-Time KPI Engine:** Dynamic telemetry updating key corporate operational metrics instantly with every decision:
   * **Cost Control** (OpEx efficiency)
   * **Delivery SLA** (On-time fulfillment rate)
   * **Risk Resilience** (Supply chain stability)
   * **Customer Satisfaction** (NPS / Client retention)
   * **Sustainability Score** (Carbon and environmental footprint)
5. **Executive Performance Review:** A final post-incident dashboard featuring an **Overall Operational Excellence Rating**, a core breakdown of systemic strengths/weaknesses, and a generated executive summary of your performance under crisis conditions.

---

## 🎨 Design & Systems Architecture

* **Enterprise Command Center UI:** A sleek, premium dark-theme control room aesthetic optimized for analytical monitoring and low cognitive load during high-stress simulations.
* **Component-Driven State Engine:** Engine constructed using functional React hooks (`useState`, `useEffect`) to orchestrate live, time-based asynchronous events and multi-metric point aggregation.
* **Fluid UI Elements:** Linear progress indicators, animated alert flashes for high-priority incidents, and smooth transition states giving immediate tactical feedback.
* **Zero Overhead Dependencies:** Packaged entirely into a single standalone HTML document using React and Babel via secure CDN pipelines, running reliably out-of-the-box offline.

---

## 🛠️ Tech Stack & Requirements

* **Runtime Environment:** Modern Web Browser (Chrome, Edge, Firefox, Safari).
* **Core View Library:** React JS (via CDN)
* **Compiler:** Babel JSX (via CDN)
* **Styling Engine:** Raw CSS3 (Flexbox, Grid, custom CSS variables, and keyframe animations).
* **Assets:** 100% self-contained codebase. No external asset or font calls required, enabling seamless offline capabilities.
